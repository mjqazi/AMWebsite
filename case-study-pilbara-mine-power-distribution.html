25 MVA mine distribution package, issued through a reset.

How a remote iron-ore operation rebuilt its 132/11 kV substation and switchgear documentation after a project reset compressed the drawing schedule.

Project Snapshot

Client: remote Australian mining operation, anonymized. Scope: 132/11 kV substation documentation, 25 MVA transformer, 11 kV switchgear, hoist winder, crusher, SAG mill, ventilation, process plant MCCs, and MicroStation-standard drawing handover.

The reset

The project was already in the field when the document control baseline was reset. Vendor packages had moved faster than the client's drawings, several loads had been reclassified, and operations needed a reliable IFC set before the next shutdown package. The schedule left no room for a traditional local drafting queue.

How we organized the package

  • Separated HV, MV, and process-load sheets so review responsibility was clear.
  • Rebuilt the 132/11 kV single-line and feeder schedule from vendor data and field markups.
  • Mapped every major load to switchgear, protection references, and drawing numbers.
  • Issued rolling review packs so the client's electrical lead could clear sections while production continued.
  • Returned MicroStation source files and PDFs aligned to the client's title-block and layer standards.

Result

48Drawing sheets
2 wkAhead of reset
0Field mods from docs

The final package supported shutdown planning without further drawing delay. More importantly, operations received a maintainable documentation baseline for future protection updates, load additions, and incident-energy reviews.

Why it mattered

Mine power systems accumulate operational changes quickly. If the single-line and feeder schedules lag the plant, every shutdown becomes a documentation exercise. The value of this engagement was not only getting 48 sheets out the door; it was giving the site a power-distribution record that operations, maintenance, and engineering could trust again.
